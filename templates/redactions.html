How to use Strata

What is the Index tab?

Run Boolean keyword searches across an entire production and export hit reports.

Getting started

1
Create a production — type a name and click +.
2
Add files — drag & drop or use Files / Folder buttons.
3
Build Index — click ⚡ Build Index.
4
Search and Export hit reports.

What is the Production tab?

Deposition prep. Search for terms, pick responsive documents, export as ZIP with responsiveness log.

What is the Redactions tab?

Scans a production for redaction markers — black bars and PDF /Redact annotations — and tells you exactly which pages they appear on. Useful for identifying what opposing counsel has withheld.

How to run a scan

1
Select a production — does not need to be indexed first.
2
Start Scan — Strata scans every PDF and Word document.
3
Review findings — page numbers and redaction type for each document.
4
Export — findings list, full report, or ZIP of redacted documents.
Note: Detects black redaction bars and official PDF /Redact annotations. Does not detect white-out on scanned documents.

Boolean operators

termination AND severance
fired OR terminated
contract NOT expired

Phrases & wildcards

"wrongful termination" — exact phrase
terminat* — wildcard
~termination — fuzzy
"fired" W/10 "email" — proximity
