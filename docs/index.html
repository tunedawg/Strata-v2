Features

Everything you need.
Nothing you don't.

Defendants bury documents deep in productions hoping you won't find them. Strata makes sure you do - without the enterprise price tag.

Boolean Search

Full Boolean syntax - AND, OR, NOT, exact phrases, wildcards, fuzzy matching, and proximity operators like W/10. Same logic as enterprise review platforms.

ESI Hit Reports

Generate formatted ESI hit reports as Excel or PDF, broken down by term, document, and page number - ready for motion practice, Golden Rule Letters, Motions to Compel, or production to opposing counsel.

Any File Type

PDF, DOCX, XLSX, CSV, EML, MSG, PPTX, TXT, and images via OCR. If it's in the production, Strata can search it.

Deposition Prep

Your co-working tool for depositions, Golden Rule Letters, Motions to Compel, and trial prep. Search by witness name or key terms, hand-pick responsive documents, and export a ZIP packet with a responsiveness log.

Redaction Scanner

Immediately see what the other side is holding back. Strata scans the entire production for redacted documents - mapping black bars and PDF redaction annotations by page - so you can quickly assess compliance or violation of the Court's orders.

Fully Local

Runs entirely on your computer - no cloud uploads, no SaaS subscriptions, no per-document charges. Your data stays on your machine.

How it works

Up and running
in minutes.

Upload any production in native format, index once, and search forever. No configuration, no IT involvement, no cloud.

1

Launch Strata

Install Strata from the Microsoft Store and launch it like any other Windows app. No separate runtime setup or command-line steps.

2

Add your documents

Click Add Files or Add Folder to browse for documents. Strata accepts PDF, DOCX, XLSX, CSV, EML, MSG, PPTX, TXT, and images. Add the whole production folder at once.

3

Build the index

Hit Build Index. Strata reads every document and builds a searchable index. Takes a few minutes for large productions - only needs to happen once.

4

Search and export

Enter your search terms and get instant results with clickable page links. Export hit reports for opposing counsel or build document packets for depositions directly from the app.

Search syntax

Powerful search.
Familiar syntax.

The same Boolean logic you'd use in Relativity or Concordance - no learning curve.

Boolean operators

termination AND severance Both terms present
fired OR terminated Either term present
contract NOT expired First without second
term AND NOT other Explicit AND NOT

Advanced matching

"wrongful termination" Exact phrase
terminat* Wildcard - matches terminate, termination...
~termination Fuzzy - catches misspellings
"fired" W/10 "email" Proximity - within 10 words
